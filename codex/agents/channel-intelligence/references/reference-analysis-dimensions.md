# Reference Analysis Dimensions

Use these dimensions when analyzing reference videos, channel materials, and source content.

## Evidence Layers

- Deterministic evidence: ffprobe metadata, scene/segment timestamps, extracted keyframes, local transcripts, OCR text, screenshots, thumbnails, and media asset manifest entries.
- Model-inferred evidence: multimodal observations, direct-video model notes, visual descriptions, inferred camera movement, inferred mood, and style labels.
- Human or Director evidence: explicit user notes, channel positioning, brand constraints, approval decisions, and manually supplied timestamps.
- Promote a rule to channel format only when its evidence layer and confidence are clear. Missing deterministic evidence can still support planning, but it must be recorded as a limitation when it affects rights, factual claims, audio timing, visual style, or source fidelity.

## Reference Video Dimensions

- Narrative structure: hook, setup, proof, conflict, reveal, payoff, CTA, recurring segments.
- Pacing: average shot length, fast/slow sections, pause points, density of ideas, rhythm changes.
- Shot grammar: shot size, angle, camera movement, transitions, B-roll categories, product/UI shots, talking head, screenshots, archival footage, AI-generated shots.
- Visual language: color palette, contrast, typography, captions, lower thirds, framing, safe areas, logo usage, recurring overlays.
- Motion language: kinetic text, zooms, pans, match cuts, wipes, light leaks, 3D moments, UI animations, data visualization, rhythm to voice or music.
- Audio language: voice style, music bed, SFX, silence, ambience, dialogue, mix density.
- Source handling: citations, screenshots, quotes, maps, charts, article excerpts, webpage scrolls.
- Packaging: title formula, thumbnail composition, opening frame, end screen, subscription CTA, metadata conventions.
- Reuse versus novelty: repeated format elements, episode-specific variations, recurring assets, unique angle.

## Web And Blog Source Dimensions

- Core claims and facts.
- Product, topic, or story entities.
- Useful visual evidence: screenshots, charts, quotes, diagrams, social posts, dates, product images.
- Claims requiring verification.
- Rights or attribution constraints.
- Narrative opportunities: contradiction, transformation, timeline, comparison, expert insight, user pain, surprising stat.

## Channel Format Dimensions

- Audience promise and content pillars.
- Hero, hub, and hygiene content fit.
- Episode structure and timing ranges.
- Brand voice, narration density, and reading level.
- Color, typography, caption, thumbnail, intro/outro, CTA, and audio systems.
- Reusable templates and assets.
- Forbidden patterns that would make outputs feel mass-produced or repetitive.
- Flex zones where each episode should differ.
