# Redundancy Risk Audit

Workflow:

1. Compare the planned video against channel format rules, recent/reference videos, and source materials.
2. Identify repeated openings, identical structure, repeated B-roll, repeated AI prompts, repeated captions, similar thumbnails, duplicated claims, and overused CTA patterns.
3. Separate healthy consistency from unhealthy sameness:
   - Healthy: recognizable voice, color, captions, structure, and source handling.
   - Risky: same topic angle, same wording, same generated visuals, same edit rhythm, or low-effort source paraphrase.
4. Add novelty requirements for the episode: new angle, new evidence, new visual metaphor, new example, new source, or new payoff.
5. Flag platform/policy risk when the plan looks mass-produced, repetitive, or insufficiently transformed from borrowed content.

Return risk level, evidence, and concrete changes needed before downstream production.
