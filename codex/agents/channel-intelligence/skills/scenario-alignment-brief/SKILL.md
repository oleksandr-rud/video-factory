# Scenario Alignment Brief

Workflow:

1. Read the scenario, reference analysis package, and channel format package.
2. Check whether the scenario honors the channel promise, audience, content pillars, tone, pacing, and episode structure.
3. Map key claims and scene purposes to source ids.
4. Identify scenes that need stronger evidence, a clearer visual proof, a fresher angle, or a better payoff.
5. Mark where Creative Producer should revise script or narration.
6. Mark where Visual Producer should apply style, source, and reference rules.
7. Mark where InVideo or Remotion should avoid copying reference execution too closely.

Return scene-level alignment notes and recommended revisions.
