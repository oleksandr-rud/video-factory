# Web Content Synthesis

Read `../../references/reference-analysis-dimensions.md` when source evidence needs structure.

Workflow:

1. Extract the central topic, audience value, and source intent.
2. Pull only video-useful claims, examples, entities, dates, visual evidence, product details, and contradictions.
3. Separate verified source facts from interpretation and unverified claims.
4. Identify possible visual evidence: charts, screenshots, quotes, UI flows, product shots, timelines, maps, or diagrams.
5. Suggest narrative angles that avoid simply reading the page.
6. Record source ids for every important claim or recommendation.

Return source findings, visual evidence candidates, claim notes, and story opportunities.
