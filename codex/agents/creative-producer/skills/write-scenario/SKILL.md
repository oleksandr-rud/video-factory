# Write Scenario

1. Read the Director brief, reference analysis, channel format, and constraints.
2. Choose a structure: hook, context, proof, payoff, CTA.
3. Draft scenes with timestamps and stable scene ids.
4. Keep narration compatible with target duration.
5. Mark factual claims that need verification and source ids when known.
6. Add initial visual intent without choosing footage.
7. Add a novelty angle so the episode follows the channel format without becoming repetitive.
